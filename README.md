# Lung Cancer Classification (ML vs Deep Learning)

This project focuses on classifying lung histopathological images using both classical machine learning and deep learning approaches.

## Overview
- Dataset: Lung histopathological images (Kaggle)
- Goal: Classify 3 types of lung conditions
- Approach:
  - Machine Learning (HOG + PCA + SVM, RF, Logistic Regression)
  - Deep Learning (MobileNetV2)

## Results
- Deep learning model (MobileNetV2) significantly outperformed classical ML models
- Strong performance on unseen test data

## Techniques Used
- Feature Engineering (HOG, Color Histogram)
- PCA (Dimensionality Reduction)
- Transfer Learning (MobileNetV2)
- Data Augmentation
